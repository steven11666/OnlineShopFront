這是一個使用 ASP.NET Core MVC 建立的簡易購物網站前台，包含以下功能：
- 店家公告與資訊
- 會員註冊 / 登入（使用 Identity）
- 商品列表與購物車
- 訂單結帳與歷史查詢
- PayPal 整合

 使用技術
- ASP.NET Core 
- Entity Framework Core
- SQL Server
- Bootstrap 5
